Contribute

There are many ways in which you can contribute to the continued development of OpenSesame. For example, you can

  • go bug-hunting and provide detailed reports of all the bugs that you encounter.

  • suggest new features. What is lacking in your opinion?

  • spread the word. Tell your colleagues, fellow students and friends about OpenSesame.

  • write documentation, a tutorial, or create an example experiment.

  • create a plug-in to add new functionality to OpenSesame, such as support for a specific device (a buttonbox, eye tracker, etc.).

  • contribute code (the OpenSesame source code is freely available)

For more information, please refer to the OpenSesame home page <http://www.cogsci.nl/opensesame>.
