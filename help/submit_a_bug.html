Submit a bug

Like all software, OpenSesame has bugs. You can help iron out the kinks by providing a detailed bug report if something appears to have gone wrong. New OpenSesame releases appear regularly, so there is a good chance that any bug that you report will be resolved quickly.

Please see http://www.cogsci.nl/opensesame for more information on how to report bugs.

If you report a bug, please provide the following information.

  • What exactly where you doing when the bug occurred? Please be specific: I was using the sketchpad, but it didn't work is not very helpful. I clicked on the line button in the sketchpad tab, and afterwards I was unable to re-open the sketchpad tab. When I tried to save the experiment, OpenSesame gave the following error: [error message here] is a fine description.

  • What did you expect to happen (i.e., what was the desired behavior of OpenSesame)?

  • What did happen (i.e., what was the real behavior of OpenSesame)?

  • Are you able to reproduce the bug?
